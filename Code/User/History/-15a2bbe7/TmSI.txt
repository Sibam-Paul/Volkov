Problem: Screensaver doesn't appear after 2.5 minutes
Solution 1: Check if hypridle is running
  pgrep hypridle
  If not running: hypridle &

Solution 2: Check if scripts are executable
  ls -l ~/.config/hypr/scripts/screensaver*
  Should show: -rwxr-xr-x (executable)

Solution 3: Test screensaver manually
  ~/.config/hypr/scripts/screensaver-launch
  Should start immediately

Problem: Screensaver doesn't exit when pressing keys
Solution: Press any key again, or:
  pkill -f "tte|Screensaver"

Problem: Screen is small/not fullscreen
Solution: Reload Hyprland
  hyprctl reload

Problem: TTE effects not showing
Solution: Check TTE is installed
  which tte
  If not: sudo pacman -S python-terminaltexteffects
